Radialpoint SafeCare, Inc.


For my third co-op term I worked at Radialpoint, a software company in Montreal. Radialpoint primarily designs tech support software for web and mobile applications. I was a member of Radialpoint's Build Team.

My day to day work included assisting developers with running their builds, which entailed debugging and configuration of build systems. My long term project was automating the process of moving virtual machine images from NAS (Network Attached Storage) to build agents (physical machines).

The project was completed with a client-server application using socket programming in Python. I also utilized WSF, NSIS and other small tools to complete the task. I was also responsible for setting up a web service for the application using the company's Apache server.
